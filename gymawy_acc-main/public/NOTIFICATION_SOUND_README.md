# إضافة صوت الإشعارات

## الخطوات:

1. **تحميل ملف صوت MP3**
   - يمكنك تحميل صوت مجاني من: https://notificationsounds.com/
   - أو استخدام أي ملف MP3 قصير (1-2 ثانية)

2. **استبدال الملف**
   - احذف الملف `notification.mp3` الحالي
   - ضع ملف الصوت الجديد باسم `notification.mp3` في مجلد `public/`

3. **اختبار الصوت**
   - عند استلام إشعار جديد، سيتم تشغيل الصوت تلقائياً
   - تأكد من أن المتصفح يسمح بتشغيل الصوت التلقائي

## ملاحظات:
- الصوت يعمل فقط عند استلام إشعارات جديدة عبر Socket.IO
- إذا لم يعمل الصوت، تحقق من إعدادات المتصفح للسماح بالصوت التلقائي
